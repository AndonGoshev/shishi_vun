# Asensreshtupeevski

Interactive React app with a welcome page and an interactive rope-pulling game.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Project Structure

- `/src/pages/WelcomePage.jsx` - Welcome page with rules modal
- `/src/pages/GamePage.jsx` - Interactive game page
- `/src/components/Modal.jsx` - Modal component for rules
- `PROJECT_REFERENCE.md` - Detailed project reference and specifications

## Features

- Welcome page with modal for rules
- Interactive rope-pulling game
- Smooth animations and transitions
